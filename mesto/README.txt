Mesto

О проекте:

Проект позволяет обмениваться фотографиями с другими пользователями.

Стек:

HTML;
CSS;
JavaScript.

Задачи для реализации на JS:

Рендер карточек при загрузке страницы;
Добавление новых карточек;
Удаление карточек;
Возможность ставить и убирать лайки;
Редактирование информации о пользователе;
Валидация форм.

Планы по доработке проекта:

Рефакторинг кода по ООП (ранее использовался функциональный подход);
Добавление API;
Сборка проекта с помощью Webpack.
